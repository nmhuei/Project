# Explain how a search operation for a key x is performed
We have a tree and find key x

search (tree, x):
    if x == tree->val :
        we find x and complete search
        
    if x > tree->val :
        we search the right node of tree (right subtree) with key x
    else : 
        we search the left node of tree (left subtree) with key x
        
    if we search fulling the tree but don't find x, tree not have a key x and we terminated process finding key x in the tree
    
# Analyze the time complexity of searching
Best : the key x is in root => The time complexity of searching is O(1)
worst :
    h is the height of the tree .We have 2 situation :
        1. the key x can be find in leaf => The time complexity of searching is O(h)
        2. the tree don't have key x => The time complexity of searching is O(h)
